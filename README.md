#  Currency Converter

A responsive **Currency Converter** web application built using **HTML, CSS, and JavaScript**. The application allows users to convert currencies in real time by fetching the latest exchange rates from a currency exchange API.

## Features

*  Real-time currency conversion
*  Supports multiple international currencies
* One-click currency swap
*  Responsive design for desktop and mobile
*  Fast and user-friendly interface
* Input validation for accurate conversions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Exchange Rate API

1. Clone this repository:

```bash

```

2. Open the project folder.

3. Open `index.html` in your preferred web browser.

No additional installation is required.
